## 2. 設計說明：
請撰寫一程式，讓使用者建立兩個矩陣，先輸入兩個正整數a、b，代表第一個矩陣為a x b矩陣，接著再輸入a x b矩陣的元素；第二個矩陣作法相同，最後輸出兩矩陣相乘的結果，同一列的矩陣元素請使用半形空格隔開，若無法相乘，請輸出「error」。

矩陣乘法：若A是mxn的矩陣，B是nxp的矩陣，則它們的乘積AB是 mxp的矩陣。
