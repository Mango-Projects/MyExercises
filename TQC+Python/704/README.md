# TQC+ 程式語言(第2版) 704 過半數
最新一次更新時間：2023-03-31 11:35:27

## 1. 題目說明：
請依下列題意進行作答，使輸出值符合題意要求。

## 2. 設計說明：
請撰寫一程式，讓使用者輸入一個正整數n（1 < n ≤ 15），接著輸入n個整數，判斷此數列中是否有數值出現的次數超過半數。若有，請輸出此數值為何；若無，請輸出「error」。

> 提示：n個整數的數列中，若出現次數大於n/2的值，稱為「過半數」。

## 3. 輸入輸出：
### 輸入說明
一個正整數n（1 < n ≤ 15）及n個整數

### 輸出說明
判斷是否有過半數

---

### 範例輸入1
```
5
2 1 3 2 2
```
### 範例輸出1
```
2
```
### 範例輸入2
```
4
5 5 6 6
```
### 範例輸出2
```
error
```
