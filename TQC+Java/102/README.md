# TQC+ 物件導向程式語言Java 102 偶數相加

## 1. 題目說明：
請開啟C:\ANS.CSF\JP01資料夾中的JPD01.java進行編寫。依下列題意進行作答：將偶數值相加，使輸出值符合題意要求。檔案名稱請另存新檔為JPA01.java，儲存於C:\ANS.CSF\JP01資料夾，再進行評分。

## 2. 設計說明：
(1) 請撰寫程式，接收輸入的兩個數字，若兩個數字皆是整數且為偶數，輸出相加結果；若輸入的數字不符合要求，請轉換為數字0。

### 輸出說明
